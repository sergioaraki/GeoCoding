GeoCoding
=========

Con este ejemplo de GeoCoding en Cascades, se puede imitar el comportamiento de un active text que al 
detectar una dirección de genera el link automáticamente a la aplicación de Mapas.
Este ejemplo permite poner una dirección con el mismo estilo de un active text y al hacerle click abre 
un Sheet con un mapa y el pin a la georreferenciación de la dirección.
